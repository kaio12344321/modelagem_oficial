Identificação

[FS01]

Função

Verificar Senha

Atores

Aluno e Bibliotecário

Fluxo Secundário

Plataforma compara as informações digitas com as informações cadastradas

 

Caso as informações forem iguais, a plataforma exibe a página inicial

 

Caso as informações forem diferentes, a plataforma exibe uma mensagem de erro

e contabiliza a tentativa

 

Identificação

[FS02]

Função

Redefinir Senha

Atores

Aluno e Bibliotecário

Fluxo Secundário

Caso a plataforma contabilize três vezes as informações diferentes, é exibido uma

mensagem pedindo para o usuário redefinir a senha

 

Será enviado um link no email cadastrado, esse link fará com que ele seja redirecionado

para uma nova página, com a funcionalidade de redefinição de senha

Função

Atores

Aluno

Fluxo Secundário

A plataforma irá buscar no banco de dados a disponibilidade do livro

 

Caso o livro esteja disponível, é exibido uma mensagem na tela com a quantidade de

livros disponíveis

Atores

Fluxo Principal

Quando um aluno pegar ou devolver um livro, siga os seguintes passos:
